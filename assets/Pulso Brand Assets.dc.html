Clearspace

Keep clearspace equal to the node diameter (≈ 20% of mark width) on all sides. Minimum size: 16px icon, 80px lockup.

Typography
Inter Semibold
Inter Medium — UI and subtitles
Inter Regular — body text. Wordmark is lowercase, weight 600, tracking −0.025em.
Color
#0a0a0a ink
#fffaf0 cream
#ff4d8b node

Ink + cream are the core pair. Hot pink is the only accent used in the mark; the extended palette is for illustration and UI states, one accent per surface.

Do
  • Use the mono (single-color) mark at 16–24px
  • Keep the node pink only when the stroke is ink or cream
  • Use the dark tile for app icons and avatars
Don't
  • Don't outline, shadow, or add gradients to the mark
  • Don't rotate the pulse or detach the node
  • Don't set the wordmark in any weight but 500–600
